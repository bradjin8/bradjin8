# Jin (@bradjin8)

**Project Manager / Full-Stack Engineer — AI-Augmented Development · TypeScript · Node.js · Mobile**

I build software with AI as a core collaborator — using agentic workflows,
RAG pipelines, and LLM-integrated tooling to accelerate development cycles,
raise code quality, and unlock capabilities that traditional approaches can't
match efficiently. My focus is on making human-AI collaboration practical and
production-ready, not experimental.

---

## Philosophy

Software development is changing faster than most teams adapt.
I treat AI agents as first-class contributors — scoping tasks for them,
reviewing their output rigorously, and designing systems that amplify
what both humans and models do best.
The goal is shorter feedback loops, higher consistency, and more time
spent on the problems that actually require human judgment.

---

## What I work on

- **Human-agentic AI systems** — agentic development workflows, LLM-driven
  automation, AI pair programming at scale; designing tasks and tooling so
  agents contribute meaningfully to real codebases rather than toy examples
- **RAG & knowledge retrieval** — hybrid dense/sparse vector search (Pinecone),
  context assembly pipelines, domain-specific retrieval for AI assistants
- **Backend & infrastructure** — TypeScript / Node.js plugin architectures,
  REST + WebSocket APIs, SQLite / PostgreSQL, Docker, Nginx, TLS,
  authentication (TOTP 2FA, server-side sessions)
- **Cross-platform apps** — React Native (iOS & Android), desktop with Electron,
  native iOS in Swift, Android with Java/OpenGL ES
- **Blockchain & Web3** — Solana NFT tooling, Ethereum / Polygon smart contracts
  (Solidity), DEX on EOSIO / WAX, multi-chain NFT marketplaces
- **C++ & systems** — Boost library research and hands-on work,
  low-level data modelling
